==== 6.(em) ====
get parent font size * em In CSS, the `em` unit is a relative measurement used primarily for font sizes, but it can also be applied to other properties like spacing and layout. The value of `1em` is equal to the current font size of the element. For example, if the font size of the parent element is 16 pixels, then `1em` is also 16 pixels³. This unit is particularly useful for creating scalable and responsive designs because it adjusts based on the font size of the parent element, making it easier to maintain consistent proportions across different screen sizes and devices³⁴ .
